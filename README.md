# Projekt "Baxi" 🤖

Willkommen bei Baxi, deinem ultimativen Discord-Bot mit einer Vielzahl von Features! 🚀
Komplett grstis, ohne Features die nur für premium sind. 

## 🌐 Überblick

Baxi setzt den Schwerpunkt auf Sicherheit und soziale Interaktion, um deinen Discord-Server optimal zu schützen und aktiv zu halten.

## 🔒 Sicherheit geht vor!

Dank des KI-Chatfilters und der Spammer-Datenbank ist es nahezu unmöglich, bösartigen Content auf deinem Server zu finden. Baxi arbeitet hart daran, deinen Server sicher und geschützt zu halten! 🛡️

## 🎮 Minispiele und Globaler Chat

Halte die Community aktiv und unterhaltsam mit den Minispielen und dem Globalen Chat von Baxi! Lass die Mitglieder Spaß haben und miteinander interagieren. 🌍🎲

## 🛡️ Verifikationssystem

Schütze deinen Server vor Bot-Accounts mit dem Verifikationssystem von Baxi. Stelle sicher, dass nur echte Benutzer Zugriff haben und halte die Bots fern! 🚫🤖

## 👋 Willkommenssystem

Begrüße automatisch alle neuen Benutzer auf deinem Server und zeige ihnen den Weg. Lass Baxi die Arbeit übernehmen und sorge für eine herzliche Atmosphäre! 🎉
